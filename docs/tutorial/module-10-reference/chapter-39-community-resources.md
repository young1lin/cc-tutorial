## 39.10 贡献方式

### 贡献代码

```bash
# Fork 仓库
git clone https://github.com/YOUR_USERNAME/claude-code.git

# 创建分支
git checkout -b feature/your-feature

# 提交更改
git commit -m "Add your feature"

# 推送分支
git push origin feature/your-feature

# 创建 Pull Request
```

### 报告问题

1. 搜索现有 Issues
2. 创建新 Issue
3. 提供详细信息：
   - Claude Code 版本
   - 操作系统
   - 复现步骤
   - 错误消息

### 改进文档

1. Fork 文档仓库
2. 编辑文档
3. 提交 Pull Request

### 分享经验

- 写博客文章
- 创建 YouTube 视频
- 在社区分享
- 参与 Discord 讨论
