# 单集脚本模板

> 每集视频遵循以下标准化格式

---

## 元信息

| 字段 | 内容 |
|------|------|
| 所属层级 | Layer XX |
| 预计时长 | X-Y 分钟 |
| 目标观众 | 初学者/中级/高级 |
| 前置知识 | XXX |

---

## 1. 开场（30-60 秒）

**目标**：让观众知道"这是什么"和"为什么要学"

### 脚本要点

- [ ] 一句话说明本集主题
- [ ] 一句话说明学习价值（解决什么问题）
- [ ] 可选：与其他集的关联

### 示例

```
今天我们来讲 XXX。

如果你曾经遇到过 YYY 的问题，这集就是为你准备的。

学完这集，你将掌握 ZZZ。
```

---

## 2. 概念讲解（2-3 分钟）

**目标**：建立心智模型，配合图表/动画

### 脚本要点

- [ ] 核心概念定义
- [ ] 为什么这样设计
- [ ] 与已知概念的类比
- [ ] 可视化图示（可选）

### 图表建议

- 流程图
- 对比表格
- 架构图
- 时间线

---

## 3. 实机演示（5-8 分钟）

**目标**：让观众跟着操作

### 脚本要点

- [ ] 准备工作（环境、前置条件）
- [ ] 分步骤演示
- [ ] 每步说明"做什么"和"为什么"
- [ ] 常见错误提示
- [ ] 验证成功的方法

### 演示清单

| 步骤 | 操作 | 预期结果 | 备注 |
|------|------|----------|------|
| 1 | ... | ... | ... |
| 2 | ... | ... | ... |

---

## 4. 总结（1-2 分钟）

**目标**：强化记忆，给出行动指南

### 脚本要点

- [ ] 回顾 3 个关键要点
- [ ] 什么时候用 / 什么时候不用
- [ ] 下一集预告

### 总结模板

```
今天我们学了：
1. XXX
2. YYY
3. ZZZ

记住：AAA

下一集我们讲 BBB。
```

---

## 5. 练习（可选）

**目标**：巩固学习效果

### 练习题

1. **基础**：完成 XXX
2. **进阶**：尝试 YYY
3. **挑战**：实现 ZZZ

---

## 制作检查清单

### 录制前

- [ ] 脚本已审核

- [ ] 图表已制作

### 录制后

- [ ] 画面可读
- [ ] 时长合适
- [ ] 无明显错误

### 发布前

- [ ] 字幕已添加
